2019年12月、ラオスに7泊8日の旅行に行った体験を元に、旅行記とおすすめの宿についてまとめた簡単な情報サイトを作りました。特に首都ビエンチャンとバンビエンに旅行に行かれる予定がある方はぜひご覧ください。

ラオスってどんな国？

ラオスはメコン川が横断する東南アジア唯一の内陸国で、山岳地帯、フランス植民地時代の建築物、山岳民族の集落、仏教の僧院などで知られています。首都ヴィエンチャンにある仏塔タートルアンには、仏陀の胸骨が収められているとも言われています。

歴史的な建造物は多いものの、それほど観光スポットが豊富ではないラオスですが、豊かな自然に囲まれてリラックスして過ごしたい方には最適な場所です。また、食文化はお隣タイから影響を受けており、新鮮なハーブをふんだんに使用した料理の数々もおすすめです。タイ料理ほどスパイシーではないので、お子様連れの方にも自信を持っておすすめできる場所と言えるでしょう。

おすすめゲストハウス

管理人が宿泊したゲストハウス、ドミトリーをご紹介いたします。コストパフォーマンス、快適さのバランスで選びました。

  • ビエンチャン

    • ビエンチャンバックパッカーズホテル（Vientiane Backpackers Hotel）

      メコン川近くのチャンアヌウォン公園の近くに位置しています。ヨーロッパ系の観光客が多い印象のバックパッカーです。フロントも英語でスムーズに意思の疎通ができました。カヤックやパラセイリングなどのアクティビティを予約することもできるので、ぜひ試してみてください。

      住所 100 Mixay,Norkeokhummaen Road,Chanthabuly District, ビエンチャン
      料金 560円〜
      朝食 有料。150円ほどで名物のラープなどが食べられる。
      設備 共用スペース / キッチン/ ダイニングルーム / レストラン / ランドリー(有料)

    • バックパッカーズガーテン（Backpackers Garten)

      こちらもメコン川流域の近くのバックパッカーになります。すぐ近くにはナイトマーケットがあり、地元の民芸品やおみやげを安く買うことができます。管理人はバンビエンまでのバスのチケットをこちらで購入しました。

      住所 028 Sihom Road, Vientiane City Center, ビエンチャン
      料金 500円〜
      朝食 無料。パンとスクランブルエッグ、またはオムレツを選ぶことができる。
      設備 共用スペース / キッチン / ダーツ・ビリヤード / テラス / レストラン / ランドリー(有料)

  • バンビエン

    • バンビエンフリーダムビュー（Vang vieng Freedom View）

      バンビエンのメインストリートに面しており、買い物やレストランも近くにたくさんあり、好立地です。ルアンパバーンやビエンチャン行きのバスの予約ができて、ネットで予約するよりも安いです。無料で提供される朝食のフランスパンは食べ放題です。

      住所 Kangmuong Street, 10001 バンビエン
      料金 600円〜
      朝食 無料。メインを6種類から選べる。パンはおかわり自由。
      設備 共用スペース / ダーツ・ビリヤード / テラス / レストラン / ランドリー(有料)
